滑動圖片畫廊

Swiper 是一個觸控滑動效果插件，支援多種輪播圖樣式，可設置自動播放、導航按鈕等功能。

Swiper官網

範例程式碼

 const swiper = new Swiper('.swiper', {
    slidesPerView: 1, // 每次顯示的幻燈片數
    spaceBetween: 10, // 幻燈片之間的間距
    navigation: {
        nextEl: '.swiper-button-next',
        prevEl: '.swiper-button-prev',
    },
    scrollbar: {
        el: '.swiper-scrollbar',
        hide: true,
    },
});
